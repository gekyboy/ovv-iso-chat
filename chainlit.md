# OVV ISO Chat v3.2

Benvenuto in **OVV ISO Chat**, il tuo assistente intelligente per documenti ISO-SGI.

## Come usarlo

Fai una domanda sui documenti ISO (PS, IL, MR, TOOLS) e riceverai una risposta basata sulla documentazione aziendale.

### Comandi speciali

| Comando | Descrizione |
|---------|-------------|
| `/teach MR-10_01` | Spiega come compilare un documento |
| `/memoria fact ...` | Salva un fatto (es. acronimi) |
| `/memoria preference ...` | Salva una preferenza |
| `/glossario` | Mostra/cerca nel glossario |
| `/status` | Stato sistema e ruolo utente |

### Comandi avanzati (solo Admin/Engineer)

| Comando | Ruolo | Descrizione |
|---------|-------|-------------|
| `/global add fact ...` | Admin | Aggiunge memoria globale |
| `/memorie <user_id>` | Admin/Eng | Visualizza memorie utente |

## Feedback

Dopo ogni risposta, usa i pulsanti **Utile** o **Non utile** per aiutare il sistema a migliorare!
